3D Models

Diorama

This is a diorama with the main subject being the Japanese youkai, karakasa kozou. This piece features a short animation that makes use of basic rigs, as well as a special effect to give the diorama the effect of a bold, cartoonish outline around the models.

On top of the model are short captions added to explain the model. The intentions behind the inclusion of each the diorama can be viewed using the link above.

Virtual Environment

This is a playable Unity project, with all models being made by myself. It is a piece to describe various parts of my personal life.

The primary intention was to be somewhat surreal, with some clashing aesthetics in the environment throughout the playthrough. It also has some rooms featuring my own personal hobbies as the theme.
